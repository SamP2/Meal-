# Mess Owner - Quick Start Guide

## 🚀 For Mess Owners

### Step 1: Login
1. Open the app
2. Tap "Owner Login" (if available)
3. Enter your email and password
4. Tap "Login"

### Step 2: Register Your Mess (First Time)

**You'll see an empty dashboard with "Register Mess" button**

1. **Tap "Register Mess"**

2. **Fill in the form:**
   ```
   Mess Name: [Your mess name]
   Area: [Your area, e.g., Kothrud, Pune]
   Food Type: [Tap Veg or Non-veg]
   Price Range: [Optional, e.g., ₹80-120]
   ```

3. **Detect Location:**
   - Tap "Tap to detect location"
   - Allow location permission
   - Wait for "Location detected" ✅

4. **Tap "Register Mess"**

5. **Success!** Your mess is now registered

### Step 3: Update Today's Menu

**From the dashboard:**

1. **Tap "Update Menu"** on your mess card

2. **Select meal type:**
   - Tap "🍛 Lunch" or "🍽️ Dinner"

3. **Add items:**
   ```
   Item name: [e.g., Dal Rice]
   Price: [e.g., 60]
   Tap "+ Add Item"
   ```

4. **Repeat** for all items

5. **Tap "✅ Save Menu"**

6. **Done!** Your menu is now live

### Step 4: Open Your Mess

**From the dashboard:**

1. **Tap the status badge** (CLOSED/OPEN)
2. Status changes immediately
3. Students can now see your mess!

## 📱 Daily Routine

### Morning (Before Opening)
1. Open app
2. Tap "Update Menu"
3. Add today's lunch items
4. Save menu
5. Tap status badge to mark as OPEN

### Evening (Before Dinner)
1. Open app
2. Tap "Update Menu"
3. Switch to "Dinner"
4. Add dinner items
5. Save menu

### Night (After Closing)
1. Open app
2. Tap status badge to mark as CLOSED

## 🎯 Quick Tips

### ✅ Do's
- Update menu daily
- Keep prices accurate
- Mark as CLOSED when not serving
- Use clear item names (e.g., "Dal Rice" not "DR")

### ❌ Don'ts
- Don't forget to mark as OPEN
- Don't leave menu empty
- Don't use confusing names
- Don't forget to update prices

## 🆘 Common Issues

### "Location not detected"
**Solution:**
1. Check if location is enabled on phone
2. Allow app to access location
3. Try again
4. If still fails, go outside for better GPS signal

### "Failed to register mess"
**Solution:**
1. Check internet connection
2. Make sure all required fields are filled
3. Try again
4. Contact support if issue persists

### "Menu not saving"
**Solution:**
1. Make sure at least one item is added
2. Check item name is not empty
3. Check price is a valid number
4. Try again
